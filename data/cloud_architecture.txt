Cloud Architecture Overview

This document describes the architecture of the web application deployed in the cloud.

1. Frontend
The frontend is built using React and deployed on a CDN for fast global delivery.

2. Backend
The backend is implemented using Node.js and Express. It runs inside Docker containers.

3. Database
The system uses PostgreSQL as the primary database.
Redis is used for caching to improve performance.

4. Authentication
JWT (JSON Web Tokens) are used for stateless authentication.
Passwords are hashed using bcrypt.

5. Deployment
The application is deployed using Kubernetes.
Auto-scaling is enabled based on CPU usage.

6. Monitoring
Prometheus collects metrics.
Grafana is used for visualization.
